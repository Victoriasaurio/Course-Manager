# Compile program
**Verify that you have installed NodeJS and check your version, for this proyect was used v10.21.1**
``` bash
node -v
```
**Installing all dependencies**
``` bash
npm i
```
**Installing MongoDB**
``` bash
sudo apt update
sudo apt install -y mongodb
```
**Verify mongo's service**
```bash
sudo systemctl status mongodb
```
***Using Docker***
```bash
docker start id
```
